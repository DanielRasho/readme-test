## Como Correrlo

1. Crear archivo de diccionario. con la siguiente sintaxis
   
   ```bash
   # diccionario.txt
   [mujer, girl]
   [mover, move]
   ```

2. Compilar los archivos

```bash
javac main.java
```

2. Correr compilados

```bash
java main.class
```
